Project Idea

Overview

The project will be a smartphone software that is used to track the price of items that are currently being sold and bought around the world. It is an app that will mainly be used to help secondhand sellers price their item by displaying the retail price of the item it currently is at and display information regarding the item, including a tracker of cost over the years or recent prices it has been sold at. The main target audience or secondhand merchants.

Motivation

Sites like Ebay and facebook marketplace are commonly used to sell secondhand items yet they lack a means to help assist sellers in marketing the value of their items efficiently. Sellers on these sites have to research the cost and value of their item before creating a listing which is immensely inefficient. Those that spend a big chunk of their time researching what they are planning to sell and make a majority of their income from working by selling in these types of marketplaces will find my app to be extremely useful as they will have a detailed organised set of information along with a pricing guide to assist them.

Description

This smartphone software is intended to support merchants on websites such as Facebook Marketplace, Ebay and gumtree and other similar platforms. The software is designed to be simple and responsive and is intended to run codes that are able to obtain data from databases which contain the necessary statistics which is then organised and structured out to display information about a massive range of items that have been sold on platforms that offer the ability to sell second hand items.
Price Tag Feature
The main feature of this software will be a price tag displayed on every item possible as well as an extended feature to take an in-depth look into all types of prices that it has been sold at and the ability to categorically display the data in forms such as highest price, or recent price. The price itself will have its own set of information that will display the date it was sold, the platform that it was sold at and the condition of the item at the time of trade. There will be a display of its current retail price, if it is still circulating in a market, and a visual representation of the avg price of the item over the years by providing a chart to read.
Additional Information Feature
Along with the main feature the app will provide support features that enhance the experience of the app for users. With the main focus being information intended for second hand merchants this feature will present information that will help the user obtain information about what the item is about such as what its features are, what the purpose of it is and anything that can help the user create a listing efficiently. There will be an organised section that the user can read through that allows the user to learn more about their item that they are looking to sell. The organised information will have information such as a description tab that provides a small worded description that the original manufacturer listed, a model tab to compare to other models before it or even maybe a current market tab that shows where it is currently being sold etc.
Responsive and Simple Search
By including a responsive and easy to use search method, users of the software will be able to quickly and more efficiently get to where they want whilst using the software. The search method will first allow them to search up the name of their item, or in the scenario that they do not know what their item name is they will be able to search up keywords such as a description of what their item is aka “charger” or even to type in a serial code that will potentially present the item to them. This search bar will contain the ability to filter out undesired or even desired aspects of the item the user is looking to search.

Tool and Techonolgies

To begin designing the software an integrated development environment is necessary to start this project, along with a computer/laptop and mobile phone that is running on preferably android in order to run and test programs. To create the software it is also extremely helpful but not necessary, to have a database on specific item prices, descriptions of items or even access to ebay or facebooks marketplace APIs as it will provide analytics that may be of some use to building the software.

Skills Required

In terms of software and hardware, they are easy to get yours hands on as everyone is expected to have both a laptop/computer and phone in this day and age. As for the skills required to design a project like this it will require a vast experience that I currently do not have. The developer of this software will be required to be proficient in a suitable coding language as well have the mathematical aptitude and problem solving skills. Not only are skills not as feasible, the option to use a database or APIs that are able to provide the data required to make this program successful will be difficult to obtain or create.

Outcome

If this project is successful, there may potentially be a big spike in the use of consumer-to-consumer selling platforms, this will impact the business of such platforms to have an influx of both merchants and items being sold on their platform. The original problem of second hand merchants inefficiently listing items, and wasting a lot of time to find a suitable price will not impact them as they will have the information ready within the minute. It is expected that merchants that sell second hand items as a source of income will be able to sell more if they spend less time organising the listing, and more time obtaining items that they can resell.
